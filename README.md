# DO4ME Platform - MERN Stack

A freelance marketplace platform connecting clients with freelancers.

## Features

- User Authentication & Authorization
- Task Management System
- Proposal System
- Payment Integration
- Real-time Notifications
- Wallet System

## Tech Stack

- **Frontend**: React, Bootstrap, React Router
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT
- **Real-time**: Socket.io

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
